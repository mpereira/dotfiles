# dotfiles

[mpereira](https://github.com/mpereira)'s configuration files.

## Installation
    $ git clone git://github.com/mpereira/dotfiles ~/.dotfiles
    $ cd ~/.dotfiles
    $ export LASTFM_PASSWORD='password'
    $ export LIBREFM_PASSWORD='password'
    $ rake sync
